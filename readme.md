# chaos 快速开发架构

## 快速开始

### chaos-chloud
cd /chaos-cloud
mvn install
mvn deploy
特性列表
1,开启async,duboo,nacos,redis,scheduling
2,MU,DTO,DATA定义
3,异常机制
4,拦截机制
5,多入口登录(微信小程序登录,后台登录)
6,通用分页模型
7,通用结果模型
8,DATA<->MuModel 转换
### chaos chaos-vue
cd /chaos-vue/chaos-vue
npm run lib
npm publish
cd /chaos-vue/chaos-data
npm publish
