# 开发环境
## nodejs环境(windows)
```shell script
download&install https://nodejs.org/zh-cn/download/
node -v
npm -v
npm config set registry https://registry.npm.taobao.org
```
## vue环境
```shell script
npm i webpack webpack-cli -D
webpack -v

npm i loadsh --save

npm i @vue/cli -g
vue -V

vue create xxx
```
