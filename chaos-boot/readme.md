##使用
1. pom.xml引入
```
 <parent>
        <groupId>com.cui.tech</groupId>
        <artifactId>chaos-parent</artifactId>
        <version>2.1.0-SNAPSHOT</version>
        <relativePath/>
 </parent>
 <dependencies>
    <dependency>
        <groupId>com.cui.tech</groupId>
        <artifactId>chaos-spring-boot-starter</artifactId>
    </dependency>
 </dependencies>
```
2. 代码生成
```
chaos-code run CodeGenerator
输入 firepongo 
输入 fo_user,fo_product
```
